# Angular Components Practice

A beginner-friendly Angular project built to learn **Angular Components** and **Tailwind CSS**.

## 🚀 Tech Stack

- Angular
- TypeScript
- Tailwind CSS
- HTML
- CSS

## 📦 Components

- 🧭 Navbar
- 🚀 Hero
- 📄 Footer

## 📂 Project Structure

```text
src/
├── app/
│   ├── app.html
│   ├── app.ts
│   └── app.css
│
├── components/
│   ├── navbar/
│   ├── hero/
│   └── footer/
```

## ▶️ Getting Started

```bash
git clone https://github.com/your-username/your-repository-name.git
cd your-repository-name
npm install
ng serve
```

Visit: `http://localhost:4200/`

## 🎯 This project was created to practice:

- Angular component creation
- Component-based architecture
- Basic Tailwind CSS styling

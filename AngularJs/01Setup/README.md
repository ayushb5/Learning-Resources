# Angular Setup with Tailwind CSS

## What is Angular?

Angular is a TypeScript-based frontend framework developed by Google for building single-page web applications (SPAs).

---

## Prerequisites

- Node.js
- npm

Check versions:

```bash
node -v
npm -v
```

---

## Install Angular CLI

```bash
npm install -g @angular/cli
```

Verify installation:

```bash
ng version
```

---

## Create a New Angular Project

```bash
ng new project-name
```

Choose **Tailwind CSS** as the stylesheet when prompted.

---

## Navigate to the Project

```bash
cd project-name
```

---

## Run the Development Server

```bash
ng serve
```

or

```bash
npm start
```

Open in your browser:

```
http://localhost:4200
```

> **Note:** Angular runs on **port 4200** by default.

---

## Build the Project

```bash
ng build
```

---

## Useful Commands

| Command | Description |
|---------|-------------|
| `ng new project-name` | Create a new Angular project |
| `ng serve` | Start the development server |
| `ng build` | Build the project |
| `ng generate component component-name` | Generate a new component |
| `ng version` | Check Angular CLI version |

---
